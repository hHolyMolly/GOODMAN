# StayClean
Вёрстка многостраничного интернет-магазина - StayClean. Реализованно множество интересных решений в данной вёрстке, множество анимаций и работы с JavaScript

Используемые технологии:

- HTML/CSS;
- JS;
- Gulp;
- Swiper;
- Icons Font;
- Pixel Perfect;
